# 你的年度关键词 · Deno Deploy

一个基于前端静态页面的年度关键词小工具，支持在 Deno Deploy 上一键部署。

## 功能概览
- 输入昵称并选择性别，生成专属关键词（基于昵称的确定性生成，保证同名同性别稳定输出）。
- 两页式体验：填写页与结果页。
- QQ 群快捷跳转与复制群号弹窗。
